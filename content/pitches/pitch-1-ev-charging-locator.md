# How It Works

### For Citizens:
1. **Search** your location on the map
2. **Vote** for a new charging station location
3. **See** real-time popularity of locations
4. **Get updates** when your location is planned

### For Municipalities:
1. **Dashboard** with demand heatmap
2. **Priority scores** per location
3. **Export data** to planning systems
4. **Budget optimization** tools

### For Government:
1. **Visualize** most popular locations nationwide
2. **Identify** demand hotspots across all regions
3. **Distribute funds** equitably throughout the Netherlands
4. **Prevent concentration** in the Randstad - ensure rural & peripheral areas get their fair share

# Key Features

### Public Interface
- Interactive map of the Netherlands
- Upvote/downvote system for locations
- Comment function for specific requests
- Existing charging stations visible
- Filter for fast chargers vs. regular

### Government Dashboard
- Real-time demand visualization
- Demographic data integration
- Grid capacity overlay (collaboration with grid operators)
- ROI calculations per location
- Prioritization algorithm

### Data Integration
- Connection with RDW (vehicle registrations)
- Grid operator data (capacity)
- CBS data (demographics, housing)
- Existing charging station databases
